# CLAUDE.md

This repository is for `whileone.se`, a lightweight website for a tech consulting firm.

## Mission
Build a simple, modern, very fast website that can be hosted for free on GitHub Pages with custom domain `whileone.se`.

## Non-negotiables
- Keep it static and lightweight.
- Prefer semantic HTML and modern CSS.
- Use minimal vanilla JS only when it materially improves UX.
- Avoid large libraries and unnecessary build complexity.
- Optimize for speed, readability, and easy long-term maintenance.

## Architecture guidance
- Start with plain static files.
- Introduce tooling/frameworks only with a strong business or maintenance reason.
- Ensure GitHub Pages compatibility for deployment.
- Keep setup and edits friendly to non-specialist maintainers.

## Messaging guidance
- Communicate practical business outcomes.
- Use clear service language and concrete benefits.
- Keep tone expert, trustworthy, and concise.
- Each page should have one primary CTA.

## Minimum quality checklist
- Responsive and mobile-first.
- Semantic structure and accessible interactions.
- Good contrast and meaningful alt text.
- Basic SEO metadata (title, description, OG).
- No console errors.
